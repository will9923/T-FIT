Simula o recebimento de uma mensagem enquanto o app está aberto.

Como testar no Real?

  1. Clique no botão "Inicializar" acima.
  2. Copie o Token Gerado.
  3. Vá no Console do Firebase.
  4. Selecione Cloud Messaging -> Nova Campanha -> Notificações.
  5. Clique em Enviar mensagem de teste.
  6. Cole o token e clique em Testar.
